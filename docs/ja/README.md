# Competitive Taiko Rating

**言語:** [한국어](../README.ko.md) | [日本語](./README.md) | [English](../README.md)

太鼓の達人コミュニティにおける大会およびマッチの成績を基に実力を算出する、**C-Rating (Competitive Rating)** システムです。

---

## 📋 プロジェクト概要
既存のパフォーマンスベースのレーティングシステムとは一線を画す、**大会成績中心のレーティングモデル**を目指しています。

*   **P-Rating (Performance Rating)**: 既存のプレイデータを基に算出する実力指標。
*   **C-Rating (Competitive Rating)**: [ドンだーひろば](https://donderhiroba.jp)のオンライン大会成績を基に算出する実力指標。大会成績が高いほど、より高いレーティングが算出されます。

## 🏆 仕組み: オンライン大会
公平な実力測定のため、短い周期でオンライン大会を運営します。

*   **大会進行**: 複数のオンライン大会を開催し、成績を統合的に収集します。
*   **課題曲選定**: 公平性を維持するため、定期的に初期化される3つのグループ（A, B, C）からランダムに楽曲を抽出します。

## 📈 レーティング算出ロジック
C-Ratingは、体系的なグループ化と統計的アルゴリズムを用いて計算されます。

### 1. プレイヤーのグループ化 (Grouping)
レーティングの識別能力を確保するため、以下のルールに従ってプレイヤーをグループ化します。
*   **基準**: P-Ratingで並び替えた後にグループ化。
*   **均衡の維持**: グループ内の最高/最低レーティング差が特定の閾値内に収まるように管理。
*   **制限事項**: 1グループあたり最低2名、最大8名まで。
*   **例外処理**: どのグループにも収まらなかったプレイヤーは、レーティング差が最も小さいグループに統合。

### 2. レーティング更新 (Glicko Algorithm)
グループ化されたデータを基に、**Glickoアルゴリズム**を使用してレーティングを更新します。
*   **相対評価**: グループ内の全プレイヤーとの対戦結果をシミュレーションして更新。
*   **勝敗判定**: 自分より順位が高いプレイヤーには敗北、低いプレイヤーには勝利したものとみなし、相対的な実力を算出。
