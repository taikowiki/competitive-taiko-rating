# 案内
## 紹介
本サービスは、[ドンだーひろば](https://donderhiroba.jp)の大会記録に基づき、太鼓の達人プレイヤーの腕前をレーティングとして計算し提供します。

## 参加方法
本サイトの[メインページ](/)で確認できるドンだーひろばの大会に参加して記録を残すと、セッション終了後に自動的にレーティングが計算されます。

## レーティング算出ロジック
セッションに参加したプレイヤーのうち、自分と近いレーティングの20人と競い合い、スコアに基づいてGlicko-2アルゴリズムでレーティングを更新します。
